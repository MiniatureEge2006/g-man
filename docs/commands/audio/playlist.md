# Hybrid Group Command: playlist

## Description:
Manage your playlists.

## Subcommands:
* [playlist create](./playlist/create.md) -> Create a music playlist.
* [playlist add](./playlist/add.md) -> Add a track to an existing playlist.
* [playlist view](./playlist/view.md) -> View the contents of a playlist.
* [playlist list](./playlist/list.md) -> List your current playlists.
* [playlist play](./playlist/play.md) -> Play a playlist.
* [playlist stream](./playlist/stream.md) -> Stream a playlist without downloading to disk.
* [playlist remove](./playlist/remove.md) -> Remove a track from a playlist by its position.
* [playlist delete](./playlist/delete.md) -> Delete an existing playlist.
