# Audio commands

This category consists of Music commands.

* [join](./audio/join.md) -> Join a voice channel.
* [leave](./audio/leave.md) -> Leave the current voice channel.
* [play](./audio/play.md) -> Play a track or playlist from a given URL. Any URL that yt-dlp supports also works.
* [stream](./audio/stream.md) -> Stream a track or playlist from a given URL without downloading to disk.
* [seek](./audio/seek.md) -> Seek to a specific position in the currently playing track.
* [goto](./audio/goto.md) -> Skip to a specific track in the queue by index.
* [move](./audio/move.md) -> Move a track to another position in the queue.
* [remove](./audio/remove.md) -> Remove a track from the queue by index.
* [volume](./audio/volume.md) -> Adjust the playback volume (0-100%).
* [nowplaying](./audio/nowplaying.md) -> Display information about the currently playing track.
* [shuffle](./audio/shuffle.md) -> Shuffle the current queue/playlist.
* [repeat](./audio/repeat.md) -> Repeat the current playing track or playlist.
* [queue](./audio/queue.md) -> Display the current queue.
* [skip](./audio/skip.md) -> Skip the currently playing track.
* [stop](./audio/stop.md) -> Stop the currently playing track and clear the queue.
* [clear](./audio/clear.md) -> Clear the queue.
* [pause](./audio/pause.md) -> Pause the currently playing track.
* [resume](./audio/resume.md) -> Resume the currently paused track.
* [musicprofile](./audio/musicprofile.md) -> View your or someone else's music listening profile.
* [playlist](./audio/playlist.md) -> Manage your playlists.
