# Hybrid Subcommand: random

## Description:
Show a random tag.

## Usage:
    g-tag random [args] [personal=False]

## Examples:
    g-tag random
    g-tag random testing
    g-tag random --personal
    g-tag random --hide-name
    g-tag random --user miniatureege2006
    g-tag random --personal --hide-name
    g-tag random --user miniatureege2006 --hide-name
