# Hybrid Group Command: tag

## Description:
Tag management commands.

## Aliases:
    t

## Usage:
    g-tag <name> [args]

## Examples:
    g-tag test hi

## Subcommands:
* [tag create](./tag/create.md) -> Create a tag.
* [tag show](./tag/show.md) -> Show a tag.
* [tag remove](./tag/remove.md) -> Remove a tag.
* [tag edit](./tag/edit.md) -> Edit an existing tag.
* [tag raw](./tag/raw.md) -> Get the raw content of a tag.
* [tag info](./tag/info.md) -> Get information about a tag.
* [tag search](./tag/search.md) -> Search for tags by name or content.
* [tag rename](./tag/rename.md) -> Rename a tag.
* [tag transfer](./tag/transfer.md) -> Transfer ownership of a tag (server or personal).
* [tag list](./tag/list.md) -> List available tags.
* [tag alias](./tag/alias.md) -> Manage tag aliases.
